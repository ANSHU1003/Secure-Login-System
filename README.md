# Secure Login System

A secure web application developed using Python Flask.

## Features

- User Registration
- User Login
- Password Hashing
- SQL Injection Protection
- Session Management
- Logout Feature

## Technologies Used

- Python
- Flask
- SQLite
- Werkzeug Security

## Security Features

- Hashed Password Storage
- Parameterized SQL Queries
- Session-Based Authentication
- Logout Functionality

## How to Run

```bash
pip install flask
python app.py
```
